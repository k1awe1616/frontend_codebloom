How to Play and Practice

Levels and Progression

Our platform is designed to help you master programming concepts through a series of levels. Each level focuses on a specific programming language and a unique challenge. You must complete the current level to unlock the next one.

  1. Navigate to the **"Play"** section from the main menu.
  2. Select the language and level you wish to attempt.
  3. Read the problem description carefully and write your code in the provided editor.
  4. Run your code to check for errors and to see if it passes the tests.
  5. Once you pass all tests, your score and completion time will be recorded.

Practice Debugging

Debugging is a core skill for any developer. The **"Practice Debugging"** section provides you with real-world scenarios where you must find and fix bugs in pre-written code.

  • The code will have one or more bugs that you must identify.
  • The editor will highlight potential areas of interest, but the solution is up to you.
  • Test your code after each fix to see if you have resolved the issue.
  • There is no time limit, so take your time and learn from the experience.

Checking Students Progress

You can view your performance and compare it with others on the **"Rankings"** page. For a more detailed breakdown, teachers and administrators can access the **"Student Progress"** dashboard using their credentials.
